# DJ Mode+

DJ Mode+ brings a mobile-style DJ experience to Spotify Desktop using Spicetify.

The main way to use DJ Mode+ is through a keyboard shortcut, allowing instant control without touching the UI.

## Main Activation
- **Ctrl + Space** â†’ Open DJ Mode+ from anywhere in Spotify  
  This is the **primary and recommended way** to use the extension.

## Features
- Smart DJ queue based on search input
- Play full albums using `-a`
- Play EPs using `-e`
- Play playlists from your library using `-p`
- Repeat a track using `-r`
- Open artist, album or playlist pages using `-o`
- Minimal, native-style modal interface

## Examples
- `Bad Bunny -o`
- `Un Verano Sin Ti -a`
- `Un Verano Sin Ti -ao`
- `Mis favoritas -p`
- `Monaco -r`
