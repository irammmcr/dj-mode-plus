# DJ Mode+

DJ-like smart play for Spotify Desktop using Spicetify.

## Features
- Smart DJ queue from search
- Play full albums using `-a`
- Play EPs using `-e`
- Play playlists from your library using `-p`
- Repeat a track using `-r`
- Open artist/album/playlist page using `-o`
- CTRL + Space quick launcher

## Examples
- Bad Bunny -o
- Un Verano Sin Ti -a
- Un Verano Sin Ti -ao
- Mis favoritas -p
- Monaco -r

## Installation
Install via Spicetify Marketplace.

## Requirements
- Spicetify
- Spotify Desktop
